*Git Clone

O Git clone é definido como um utilitário de linha de comando. Sua função é permitir a clonagem, ou seja, uma cópia idêntica de um repositório em rede para que o mesmo se torne um repositório local, permitindo dessa forma a integração de usuarios em um projeto.
